4.정상성 · 단위근 — ACF, PACF, ADF

시계열 분석의 출발점. 회귀·상관·VAR에 앞서 각 변수가 정상(stationary)인지 먼저 확인해야 한다 — 비정상 변수를 레벨로 회귀하면 가성회귀(spurious regression)가 된다. ACF가 천천히 감쇠하면 단위근 의심 → ADF로 확정. 비정상이면 차분하거나(VAR), 공적분을 점검한다(§10).

Sources: Dickey & Fuller (1979), JASA; MacKinnon (1996) 임계값. 평균회귀 속도는 반감기 ln(0.5)/ln(φ)로 요약. 분석 워크플로우 순서는 Enders, Applied Econometric Time Series (Wiley) 표준을 따른다.

8.VAR · 충격반응 · 분산분해 — Vector Autoregression

다변수 동태 시스템. 변수 2–4개와 시차 p를 선택하면 축약형 VAR(p)를 방정식별 OLS로 추정하고, 직교화 충격반응함수(IRF)와 예측오차 분산분해(FEVD)를 계산한다. VAR은 정상 변수에 적용하므로, §4에서 I(1)로 판정된 변수는 차분 옵션을 켜고 분석할 것.

Sources: Sims (1980), "Macroeconomics and Reality," Econometrica 48(1) — VAR의 원전; 직교화 IRF는 잔차 공분산의 Cholesky 분해를 사용하므로 변수 순서(ordering)에 의존한다 — 위에 둔 변수일수록 동시점에 외생적(Hamilton, Time Series Analysis, 1994, ch.11).

Cholesky 순서 = 선택한 변수 순서(좌→우). 순서를 바꾸려면 칩을 다시 클릭(해제 후 원하는 순서로 재선택).

Table 5. VAR 계수 — 방정식별 (행=종속, 열=설명)

충격반응함수 (Orthogonalized IRF)

각 패널은 한 변수에 1 표준편차 충격이 가해졌을 때 시스템 변수들의 동태 반응 경로다. ±2 S.E. 밴드는 해석적 근사(Lütkepohl)로 표시. 반응이 0으로 수렴하면 안정적 시스템이다.

예측오차 분산분해 (FEVD)

각 변수의 h-기 예측오차 분산을 어느 충격이 몇 % 설명하는지 분해. 자기 충격 비중이 높으면 외생적, 타 변수 비중이 높으면 시스템 내생적이다.

Table 6. FEVD — 행=대상변수, 열=충격원천 (%)

9.Granger 인과 — Granger Causality

"X의 과거가 Y의 미래 예측에 도움이 되는가"를 검정한다. 인과(causation)가 아니라 예측적 선행성임에 유의. 모든 변수쌍 방향별로 F-검정을 일괄 수행.

Sources: Granger (1969), "Investigating Causal Relations by Econometric Models and Cross-spectral Methods," Econometrica 37(3). 제한·비제한 모형의 SSR로 F = [(SSR_r−SSR_u)/p] / [SSR_u/(n−2p−1)] ~ F(p, n−2p−1). 비정상 변수는 차분 후 검정 권장(Toda–Yamamoto 대안도 존재).

Table 7. Granger 인과 검정 — 행(원인) → 열(결과), F-통계량 / p값

10.공적분 — Engle–Granger 2단계

두 비정상 I(1) 변수가 장기적으로 함께 움직이는가(공적분)를 본다. 공적분되면 레벨 관계가 가성적이지 않고, 단기 이탈은 오차수정(ECM)으로 회귀한다 — 페어트레이딩·헤지비율의 통계적 근거.

Sources: Engle & Granger (1987), "Co-integration and Error Correction," Econometrica 55(2). 1단계: Y = α + βX 회귀로 잔차 추정. 2단계: 잔차에 ADF — 단위근 기각 시 공적분. 임계값은 일반 ADF보다 엄격(Engle–Granger 임계값 사용).

Figure 6. 공적분 잔차 (장기균형 이탈) — 평균회귀하면 공적분
잔차가 0 주위로 회귀하면 두 변수는 장기균형을 공유. β = 헤지비율(Y 1단위당 X β단위).

11.방법론 · 참고문헌

분석 순서 (권장 워크플로우)

① 기술통계·분포로 데이터 성격 파악 → ② 단위근 검정으로 정상성 확인 → ③ 정상이면 레벨, I(1)이면 차분으로 상관·회귀 → ④ 다변수 동태는 VAR(정상화 후) → ⑤ I(1) 변수쌍은 공적분 점검 후 공적분되면 레벨 VECM, 아니면 차분 VAR. 이 순서는 비정상 시계열의 가성회귀를 피하기 위한 표준 절차다(Enders 2014).

구현

모든 계산은 브라우저 내 순수 JavaScript로 수행되며 데이터는 외부로 전송되지 않는다. OLS·VAR는 정규방정식 (X′X)⁻¹X′y를 가우스 소거로 풀고, t·F p값은 불완전 베타함수 연분수 전개로 계산한다. PACF는 Durbin–Levinson, ADF는 상수항 포함 사양. VAR IRF는 잔차 공분산의 Cholesky 분해로 직교화하며, FEVD는 직교화 IRF 제곱 누적비율로 계산한다.
